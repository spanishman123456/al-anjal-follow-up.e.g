# المطلوب منك فقط (٣ خطوات)

تم تجهيز المشروع. باقي عليك الآتي:

---

## ١) تجيب رابط MongoDB (مرة واحدة)

1. ادخل: **https://www.mongodb.com/cloud/atlas**
2. سجّل دخول أو اعمل حساب مجاني
3. اضغط **Build a Database** → اختار **M0 FREE** → Create
4. اعمل مستخدم: **Database Access** → Add User → username و password (احفظهم)
5. افتح الشبكة: **Network Access** → **Add IP** → **Allow from anywhere**
6. خذ الرابط: **Database** → **Connect** → **Drivers** → انسخ الـ connection string (الي يبدأ بـ `mongodb+srv://`)
7. في الرابط انسخ كلمة السر مكان `<password>` (بدون علامات)

---

## ٢) تعدّل ملف واحد في المشروع

1. افتح الملف: **backend** ثم **.env**
2. غيّر **سطر واحد فقط**:
   - بدل `MONGO_URL=mongodb+srv://USER:PASSWORD@...`  
   - حط الرابط اللي نسخته من Atlas (مع كلمة السر مكان PASSWORD)

الباقي في الملف (مثل `DB_NAME=school_db` و `JWT_SECRET=...`) يمكنك تتركهم أو تغيّرهم لو حابب.

---

## ٣) تشغيل المشروع

### لو ظهر عندك "npm غير معروف"
- استخدم الملف **`frontend\تشغيل الفرونت.bat`**: دبل كليك عليه وهو يثبت الحزم ويشغّل الفرونت (أو كليك يمين → Run).
- أو أضف Node لـ PATH مرة واحدة: ابحث في Windows عن "Environment Variables" → Edit system environment variables → Environment Variables → تحت "User variables" اختار "Path" → Edit → New → أضف `C:\Program Files\nodejs` → OK في كل النوافذ، ثم أعد فتح Cursor.

### التشغيل العادي
افتح **Terminal** في Cursor (من القائمة: Terminal → New Terminal) واكتب:

**أولاً – الباك اند (اترك هذه النافذة مفتوحة):**
```bash
cd backend
pip install -r requirements.txt
uvicorn server:app --reload --host 0.0.0.0 --port 8000
```

**ثانياً – الفرونت اند (افتح terminal جديد واكتب):**
```bash
cd frontend
npm install
npm start
```

بعد دقيقة يفتح المتصفح على **http://localhost:3000** وتقدر تستخدم التطبيق.

---

## لو حابب ترفعه على النت لاحقاً

- **الواجهة (Frontend):** سجّل على **vercel.com** واربط المشروع من GitHub واختر مجلد `frontend`.
- **السيرفر (Backend):** سجّل على **railway.app** واربط المشروع واختر مجلد `backend`، وحط نفس قيم الـ `.env` في Variables.

**ملاحظة:** لو كتبت الأوامر وظهر "pip غير معروف" أو "npm غير معروف":
- تأكد أن **Python** و **Node.js** مثبتين من الموقع الرسمي (واختر "Add to PATH" أثناء التثبيت).
- جرّب في الطرفية: `py -m pip install -r requirements.txt` بدل `pip install ...`

لو وقفت عند أي خطوة اكتب لي إيه اللي ظهر قدامك (أو صورة من الشاشة) ونكمل.
