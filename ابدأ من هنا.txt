   📝 الخطوات (3 خطوات فقط!)
═══════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────┐
│  الخطوة 1: تأكد من التثبيت                                 │
└─────────────────────────────────────────────────────────────┘

✅ Python مثبت؟ (افتح Terminal واكتب: python --version)
✅ Node.js مثبت؟ (افتح Terminal واكتب: node --version)

لو مش مثبتين:
- Python: https://www.python.org/downloads/
- Node.js: https://nodejs.org/


┌─────────────────────────────────────────────────────────────┐
│  الخطوة 2: إعداد MongoDB (مرة واحدة فقط)                  │
└─────────────────────────────────────────────────────────────┘

1. اذهب إلى: https://www.mongodb.com/cloud/atlas
2. سجّل حساب مجاني
3. Build Database → M0 FREE → Create
4. Database Access → Add User → (احفظ username و password)
5. Network Access → Add IP → Allow from anywhere
6. Database → Connect → Drivers → Python → انسخ الرابط
7. استبدل <password> في الرابط بكلمة السر
8. افتح ملف: backend\.env
9. غيّر السطر 6: MONGO_URL= (ضع الرابط هنا)


┌─────────────────────────────────────────────────────────────┐
│  الخطوة 3: تشغيل المشروع                                    │
└─────────────────────────────────────────────────────────────┘

الطريقة السهلة (دبل كليك):

1. افتح مجلد backend
2. دبل كليك على: تشغيل الباك اند.bat
3. انتظر حتى يظهر: "Uvicorn running on http://0.0.0.0:8000"
4. اترك النافذة مفتوحة!

5. افتح مجلد frontend (في نافذة جديدة)
6. دبل كليك على: تشغيل الفرونت.bat
7. انتظر دقيقة... سيفتح المتصفح تلقائياً!

❌ "pip غير معروف"
✅ استخدم ملف .bat أو اكتب: py -m pip install -r requirements.txt

❌ "npm غير معروف"
✅ تأكد من تثبيت Node.js وأعد تشغيل Cursor

❌ الباك اند مش بيشتغل
✅ تأكد من ملف backend\.env موجود ورابط MongoDB صحيح

❌ الفرونت اند مش بيشتغل
✅ تأكد إن الباك اند شغال أولاً (يجب يكون على port 8000)
